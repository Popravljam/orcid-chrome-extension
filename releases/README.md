# ORCID ID Detector - Releases

This directory contains packaged releases for distribution to browser extension stores.

## Version 1.4 (Current)

**Release Date**: December 23, 2025

**Files:**
- `orcid-id-detector-v1.4.zip` (28K) - Chrome Web Store package
- `orcid-detector-firefox-v1.4.xpi` (22K) - Firefox Add-ons package
- `RELEASE_NOTES_v1.4.md` - Release notes

**New Features:**
- Enhanced support for sites with existing ORCID integration
- Detects ORCID links with logo images
- Adds interactive magnifying glass icon next to existing ORCID logos

## Installation

### Chrome/Edge
1. Download `orcid-id-detector-v1.4.zip`
2. Extract the archive
3. Go to `chrome://extensions/`
4. Enable "Developer mode"
5. Click "Load unpacked" and select the extracted folder

### Firefox
1. Download `orcid-detector-firefox-v1.4.xpi`
2. Go to `about:debugging#/runtime/this-firefox`
3. Click "Load Temporary Add-on"
4. Select the `.xpi` file

## Store Submission

### Chrome Web Store
- Dashboard: https://chrome.google.com/webstore/devconsole
- Upload: `orcid-id-detector-v1.4.zip`

### Firefox Add-ons
