# Release Notes - Version 1.4

## New Features

### Enhanced Support for Existing ORCID Integration
- **Detects sites with existing ORCID logos**: The extension now recognizes when websites already display ORCID logo images (like `orcid_16x16.png`)
- **Adds interactive magnifying glass icon**: Places a clickable magnifying glass next to existing ORCID logos
- **Seamless integration**: Works alongside native ORCID implementations without interference
- **Quick profile access**: Click the magnifying glass to view detailed researcher information without leaving the page

## Benefits

This update is particularly useful for:
- Academic institutional repositories
- Research databases (e.g., Serbian research portal)
- University faculty directories
- Journal websites with author ORCID integration
- Any site that already displays ORCID identifiers with logos

## Technical Improvements

- Enhanced detection algorithm for ORCID links with images
- Improved mutation observer to handle dynamically loaded ORCID links
- No impact on existing plain-text ORCID ID detection
- Maintains all privacy protections and zero data collection

## Compatibility

- Chrome 88+
- Firefox 109+
- Edge (Chromium-based)

## What's Next

Visit sites like research databases to see the new feature in action. Look for ORCID logos with a green magnifying glass icon next to them!
